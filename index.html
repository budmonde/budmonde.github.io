Impression

Sep 2015

Impression is a (broken) application that was submitted for hackMIT 2015. The interprets an input image and proposes a short Youtube playlist of music that would go along with the theme of the image. The application features several cross-platform APIs in image recognition and NLP to come up with its result. The application was developed along with teammates @zhoul, @ckguo and @ajays235. AI Teacher

Fall 2015

As part of an undergraduate research experience, I worked for the Global Studies and Languages Department to produce parts of the project known as AI Teacher. This web application functions to allow foreign learners of the Japanese language a set of tools which would make the learning experience easier. The specific part I worked on featured a sentence parser with predictions on what kind of words come up most often in literature as well as provide lots of details on the syntaxical features the input sentence presented. The application can be found here.

Harmony

Jan 2015

This application was a collaborative sheet music editor with real time database syncronization which allowed users to concurrently edit the same document. Think Google Docs for music sheets... with clunky UI and cryptic syntax for parsing the music sheets. This Ruby on Rails App was my first Web Development project. Working with @zhoul and @jtxiao was a great experience in understanding the Web framework and putting a presedence on more projects to come. Site link here.
