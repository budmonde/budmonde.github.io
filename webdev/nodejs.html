Windows:
  1. Make sure you have Git Bash installed (used in previous lecture).
  2. Install Node.js v4.5.0 LTS.
  3. Open Git Bash and go to any non-system folder. I.e. "Documents", or "Desktop".
  4. Run "npm install -g express"
  5. You're good to go!
Mac/Ubuntu:
  1. Make sure you have git installed from previous lecture.
  2. Install Node.js v4.5.0 LTS.
  3. Open a terminal and run "npm install -g express"
  4. You're good to go!
